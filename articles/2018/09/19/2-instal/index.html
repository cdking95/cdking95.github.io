安装

Seeruo提供了多种安装方式，你可以根据你的情况使用；

前置要求

php >= 5.6

1.通过包管理器(Composer)安装（推荐）

SeeRuo提供了基础脚手架，可以直接用php包管理工具直接生成项目结构。

composer create-project seeruo/seeruo myblog

2.通过Git下载脚手架

你可以选择直接下载： 下载地址

也可以通过Git clone下载：

git clone git@github.com:seeruo/seeruo.git

目录结构

安装完成之后，项目的目录结构应该是

myblog
    |-App
        |-Config  // 配置文件目录
            |-config.php 
        |-Plugins // 插件目录
        |-Source  // 资源文件存放目录
        |-Themes  // 主题目录
    |-vendor

App目录名可以随便修改，比如说我有多个项目，一个博客blog，一个文档document，那么我的目录结构为

myblog
    |-blog      // 这是博客项目，修改App目录名称而来
    |-document  // 这是文档项目，完全拷贝blog的一个新项目
    |-vendor
